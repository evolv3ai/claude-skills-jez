# Claude Changelog

> Last verified: 2026-02-17

Recent changes to Claude AI and Claude Code, reverse-chronological. Focus on features that post-date training data.

## 2026-02

- **Feb 13** [Claude Code] v2.1.42 released
- **Feb 5** [Claude Code] Agent Teams (experimental) — multi-instance orchestration with shared task lists, inter-agent messaging, tmux split panes. Requires Max plan. Enable: `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`
- **Feb 4** [Claude AI] Permanent ad-free commitment announced

## 2026-01

- **Jan 26** [Claude AI] Interactive Apps launched — Slack, Canva, Figma, Box, Clay, Asana, Amplitude, Hex, Monday.com. Powered by MCP Apps protocol extension. Available on Pro, Max, Team, Enterprise.
- **Jan 2026** [Claude Code] Cowork launched — Claude Code for non-developers with local folder access. macOS only, Max plan required.
- **Jan 2026** [Claude AI] Free tier expanded — now includes file creation, connectors, skills, longer conversations
- **Jan 2026** [Claude AI] Claude in Chrome (browser agent) entered beta
- **Jan 2026** [Claude AI] Claude in Excel (spreadsheet agent) entered beta
