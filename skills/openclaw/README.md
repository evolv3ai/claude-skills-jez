# OpenClaw

Self-hosted AI gateway and orchestration platform. Routes conversations between
messaging channels (WhatsApp, Telegram, Discord, Slack) and multiple LLM providers
(Anthropic, OpenAI, Gemini, Groq, Bedrock) with tool use, session isolation, and
browser automation.

**Source**: https://github.com/coollabsio/openclaw
**Production Tested**: OpenClaw v2026.1.x on KASM 1.17.0 (Ubuntu 22.04)
